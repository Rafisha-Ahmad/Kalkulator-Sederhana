# 🧮 Kalkulator Sederhana - Flask App

Aplikasi kalkulator web sederhana berbasis **Python Flask** dengan tampilan modern menggunakan tema *Dark Mode & Glassmorphism* (efek kaca transparan). Navigasi operator interaktif menggunakan tombol pilihan langsung (tanpa dropdown).

---

## Fitur Utama
* **4 Operasi Matematika:** Pertambahan (`+`), Pengurangan (`-`), Perkalian (`*`), dan Pembagian (`/`).
* **UI Premium & Interaktif:** Efek *Glassmorphism* dengan tombol operator yang berubah warna neon menyala saat dipilih.
* **Validasi Pembagian Nol:** Mencegah aplikasi *crash* dengan menampilkan pesan error jika melakukan pembagian dengan angka `0`.
* **Fitur Reset Sempurna:** Tombol reset mengembalikan input dan membersihkan hasil kalkulasi di layar kembali seperti semula.

---

## Struktur Proyek
Pastikan struktur folder dan file kamu di komputer sudah sesuai seperti ini sebelum dijalankan:

```text
PYTHON/
 ┣ 📂 static/
 ┃ ┗ 📄 style.css        # File desain tampilan (CSS)
 ┣ 📂 templates/
 ┃ ┗ 📄 index.html       # Struktur halaman web (HTML)
 ┣ 📄 app.py             # Logika utama aplikasi (Python Flask)
 ┗ 📄 README.md          # Dokumentasi proyek
```

## Flask
* Install **pip install flask** diterminal
* Lalu jalankan dengan **python app.py**
